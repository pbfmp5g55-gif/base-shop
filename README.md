# 風松鶏ハム LP — BASE 反映マニュアル

参考画像をもとに作った 1 枚 LP。BASE.in に貼り付けて使います。

## ファイル
- `index.html` — LP 本体(CSS インライン、レスポンシブ対応)

## まずブラウザで見る
`C:\Users\yuho_shinkawa\base-shop\index.html` をダブルクリックでブラウザに表示されます。画像はプレースホルダーになっているので、実写真があれば差し替えてください。

## BASE への貼り方(2 通り)

### A. 商品ページに丸ごと埋め込む(おすすめ・最速)
1. BASE 管理画面 → 商品管理 → 該当商品 → 編集
2. 「商品説明」欄の右上「HTML」モードに切り替え
3. `index.html` の `<div class="lp-wrap">` から `</div><!-- /.lp-wrap -->` までをコピペ
4. `<style>` ブロックも一緒にコピペ(同じ商品説明欄の先頭に)
5. 保存

→ 商品ページにアクセスするとそのまま LP が表示される

### B. ショップトップを LP にする(BASE Apps「HTML 編集」が必要)
1. BASE 管理画面 → Apps → 「HTML 編集」をインストール(無料)
2. デザイン編集 → HTML/CSS 編集
3. トップページの本文部分に `<div class="lp-wrap">…</div>` をペースト
4. `<style>` は `<head>` 内 or 同じ場所の先頭に

## 画像の差し替え方法
LP 内の画像は全部 `https://placehold.co/...` のダミー URL になっています。

1. BASE 管理画面 → 商品編集 → 画像をアップロード
2. アップロード後、画像を右クリック → 「画像 URL をコピー」
3. `index.html` 内の `https://placehold.co/...` を、コピーした BASE の画像 URL に差し替え

## ブランド名・商品名を変える
`index.html` 内の以下を一括置換:
- `風松` → クライアントのブランド名
- `KAZEMATSU` → ローマ字表記
- `鶏ハム` → 商品名
- `3,500` → 価格
- `7個セット` / `200g` → 数量・容量
- `居酒屋店主が本気で作った` → キャッチコピー
- `居酒屋15年の店主が監修` などの本文 → クライアントのストーリーに合わせて

## カートボタンを動作させる
`<a href="#" class="lp-cta">` の `href="#"` を、BASE の「カートに追加する URL」or 商品ページ URL に差し替え。

商品ページ内に埋め込んだ場合は、BASE 標準の「カートに入れる」ボタンが別途あるので、この LP 内のボタンは装飾扱いでも OK。
